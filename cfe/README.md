# 1c-ai-assistant

Универсальное расширение конфигурации 1С:Предприятие (CFE) с нативным чатом ИИ-ассистента.

## Возможности

- Нативная управляемая форма чата внутри 1С, без iframe и внешних виджетов.
- Текстовые вопросы с потоковой передачей ответа.
- Прикрепление скриншотов и изображений к вопросу (требуется LLM с поддержкой vision).
- Контекстная помощь по объектам конфигурации: справочникам, документам, отчётам и т.д.
- Подключение к внешнему шлюзу ИИ через константы расширения.
- Хранение адреса шлюза, модели, интервала опроса и таймаута в параметрах расширения.

## Требования

- 1С:Предприятие 8.3.26+ (управляемое приложение, режим совместимости 8.3.26).
- Внешний шлюз ИИ, совместимый с API расширения.

## Установка

1. Собрать файл `.cfe` из исходников в каталоге `cfe/src/`.
2. Подключить расширение в конфигураторе 1С (`Конфигурация → Расширения конфигурации → Добавить`).
3. Заполнить константы расширения:
   - `ИИ_АдресШлюза` — URL шлюза ИИ, например `https://ai-gate.company.local:8443`.
   - `ИИ_ИнтервалОпроса` — интервал опроса результата в секундах.
   - `ИИ_ТаймаутЗапроса` — таймаут HTTP-запроса в секундах.
4. Промпт-подсказка задаётся каждым пользователем в форме настроек (меню «…» формы чата) и сохраняется персонально.
5. Назначить пользователям роли `ИИ_Администратор` (изменение констант) и/или `ИИ_Пользователь` (работа с чатом).

При обновлении расширения не принимайте из основной конфигурации `ИИ_ТипыОбъектовКонтекстныхЗапросов` и `ИИ_ОткрытьЧат`, чтобы сохранить адаптацию.

## MCP-сервер для 1С

Для доступа шлюза к метаданным и данным базы 1С требуется отдельный MCP-сервер, опубликованный в информационной базе по адресу `hs/mcp/rpc`.

Готовый вариант: https://github.com/vladimir-kharin/1c_mcp/

После публикации MCP-сервера в базе укажите его URL, имя пользователя и пароль в конфигурации шлюза (`mcp_servers` → `mcp_1c`).
